Beam Geometry:

The geometry of the beam is defined using a 2D analysis model where each support is defined by a node and each span is defined by an element which connects the two nodes.

Notes and exclusions:

  • This calculation form only accepts beams with rectangular cross sections, defined by the section width X section depth;
  • Only simply supported beams are supported in the current version.
